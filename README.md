# messager-flask
A messaging app made on flask

# Installation:
- git clone https://github.com/krishpranav/messager-flask
- cd messager-flask
- sudo chmod +x *

# running messager app

```
python3 -m venv env
source env/bin/activate
python3 -m pip install -r requirements.txt
python3 messager.py
```
